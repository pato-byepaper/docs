# Procedimiento Operativo Estándar para el Uso del Dashboard Técnico

<Info>
Este SOP te guía para gestionar tus tareas y mantener el hotel operando de forma eficiente usando el dashboard técnico.
</Info>

## Key Steps

<Steps titleSize="h3">
  <Step title="1 · Acceso al Dashboard Técnico" icon="gauge" iconType="solid" stepNumber={1}>
    <ul>
      <li>Inicia sesión en tu dashboard técnico.</li>
      <li>Familiarízate con la interfaz y las secciones disponibles.</li>
    </ul>
    <Tip>
      Agrega el dashboard a tus favoritos del navegador para entrar más rápido al inicio de turno.
    </Tip>
  </Step>

  <Step title="2 · Estadísticas de Rendimiento" icon="chart-line" iconType="solid" stepNumber={2}>
    <ul>
      <li>Revisa tus <Tooltip tip="Indicadores como tareas completadas, tiempos de respuesta y cumplimiento de SLA.">estadísticas de rendimiento</Tooltip>.</li>
      <li>Observa el balance entre tareas completadas y pendientes.</li>
    </ul>
    <Info>
      Si detectas caídas en rendimiento, revisa las categorías de tareas con más retrasos y ajusta prioridades.
    </Info>
  </Step>

  <Step title="3 · Gestión de Tareas" icon="list-check" iconType="solid" stepNumber={3}>
    <ul>
      <li>Mantén actualizados el estado y las notas de cada tarea.</li>
      <li>Usa <Tooltip tip="Acciones directas como crear reporte, abrir garantía o marcar avance.">acciones rápidas</Tooltip> para ganar tiempo.</li>
    </ul>
    <Warning>
      No cierres tareas sin evidencia o nota técnica. Afecta reportes y auditorías.
    </Warning>
  </Step>

  <Step title="4 · Documentación del Trabajo" icon="camera" iconType="solid" stepNumber={4}>
    <ul>
      <li>Documenta con notas y fotos el <em>antes</em> y <em>después</em> de cada intervención.</li>
      <li>Adjunta números de serie o consumibles usados cuando aplique.</li>
    </ul>
    <Info>
      La buena documentación acelera garantías y reduce retrabajos.
    </Info>
  </Step>

  <Step title="5 · Inspecciones Asignadas" icon="clipboard-list" iconType="solid" stepNumber={5}>
    <ul>
      <li>Entra a <strong>Mis pendientes</strong> para ver tareas e inspecciones asignadas.</li>
      <li>Filtra por prioridad y fecha de vencimiento para atacar primero lo crítico.</li>
    </ul>
  </Step>

  <Step title="6 · Exploración del Dashboard" icon="compass" iconType="solid" stepNumber={6}>
    <ul>
      <li>Recorre cada sección para entender todas las funcionalidades disponibles.</li>
      <li>Identifica vistas, filtros y atajos que más uses en tu día a día.</li>
    </ul>
    <Tip>
      Guarda tus filtros favoritos para repetir búsquedas en un click.
    </Tip>
  </Step>

  <Step title="7 · Gestión de Tareas Técnicas" icon="screwdriver-wrench" iconType="solid" stepNumber={7}>
    <ul>
      <li>Planifica tu día priorizando tareas por impacto y SLA.</li>
      <li>Confirma bloqueos (accesos, repuestos) antes de comenzar para evitar tiempos muertos.</li>
    </ul>
    <Check>
      Criterio de salida: todas las tareas con estado actualizado, notas claras y evidencia cargada.
    </Check>
  </Step>
</Steps>

---

## Cautionary Notes

<Warning>
Revisa tus tareas <strong>a diario</strong> para evitar pendientes acumulados.
</Warning>

<Info>
Mantén la documentación actualizada para prevenir confusiones y facilitar auditorías.
</Info>

---

## Tips for Efficiency

<Tip>
Configura recordatorios para actualizar tareas al final de cada intervención.
</Tip>
<Tip>
Usa la evidencia (notas y fotos) para mejorar la comunicación con tu equipo y acelerar aprobaciones.
</Tip>
